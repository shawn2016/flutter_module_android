# SimpleApp - 最简单的 Android 项目

这是一个最简单的 Android 项目，只有一个首页，没有导航栏。

## 项目结构

```
android_simple/
├── app/
│   ├── src/main/
│   │   ├── java/com/simple/app/
│   │   │   └── MainActivity.kt          # 主Activity
│   │   ├── res/
│   │   │   ├── layout/
│   │   │   │   └── activity_main.xml   # 主布局
│   │   │   └── values/                 # 资源文件
│   │   └── AndroidManifest.xml
│   └── build.gradle
├── build.gradle
├── settings.gradle
└── gradle/
```

## 功能

- ✅ 只有一个 MainActivity
- ✅ 简单的首页布局
- ✅ 没有底部导航
- ✅ 没有 Fragment
- ✅ 最简洁的项目结构

## 运行要求

- Android Studio Hedgehog | 2023.1.1 或更高版本
- JDK 8 或更高版本
- Android SDK 24 或更高版本（最低支持）
- Android SDK 34（编译目标）

## 如何运行

1. 使用 Android Studio 打开项目
2. 连接 Android 真机或启动模拟器
3. 点击运行按钮或使用快捷键 `Shift + F10` (Windows/Linux) 或 `Ctrl + R` (Mac)
4. 应用将自动安装并运行在设备上

## 技术栈

- Kotlin
- AndroidX
- Material Design Components
- ConstraintLayout
